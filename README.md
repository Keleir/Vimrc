# Vimrc

> My vim config both on Linux and Windows.
> 
> Kallen Ding, Feb 02 2016

###  Usage
```
git clone https://github.com/Keleir/vimrc.git
cd vimrc
git submodule update
cp -r .vimrc .vim $HOME
vim 
:BundleInstall
```
